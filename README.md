# 附录
## 全局变量
为了给业务提供尽可能多的过程及结果信息，同时还能保持其便捷性。`sl4opt`预置了下列变量，模板中可以直接引用。

| 变量name | 说明 |
| --- | --- |
| _res | 方法执行返回结果，只有成功时才有值 |
| _err | 方法执行出错信息，只有失败时才有值 |
| _time | 方法执行耗时(单位: ms) |
| _stime | 方法执行前时间戳(timestamp) |
| _etime | 方法执行结束时时间戳(timestamp) |
